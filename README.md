# АИС «Электронная библиотека»

Курсовой проект по дисциплине «Веб-разработка».
Разработал: Петрачков Владимир Владимирович, группа 241-327.

## Технологии
* **Backend:** Python (Flask)
* **Frontend:** Jinja2, Bootstrap 5, EasyMDE (Markdown)
* **Database:** MySQL 8.0
* **Infrastructure:** Docker, Docker Compose, Caddy (Proxy)

## Функциональные возможности
* **Управление книгами:** Просмотр, добавление, редактирование и удаление (права зависят от роли).
* **Жанры:** Поддержка нескольких жанров для одной книги (связь Many-to-Many).
* **Обложки:** Загрузка изображений с автоматической проверкой MD5-хэша для предотвращения дубликатов.
* **Рецензии:** Возможность оставлять отзывы с оценкой и форматированием Markdown.
* **Модерация:** Система статусов рецензий («На рассмотрении», «Одобрена», «Отклонена»).
* **RBAC:** Разграничение прав доступа для ролей Администратор, Модератор и Пользователь.

## Локальный запуск (Docker)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/L0ked/Web-Exam-2026.git
   cd Web-Exam-2026
   ```

2. Запустите контейнеры:
   ```bash
   docker-compose up --build -d
   ```

3. Приложение будет доступно по адресу: [http://localhost:5000](http://localhost:5000)
   *База данных инициализируется и наполняется начальными данными автоматически при первом обращении.*
